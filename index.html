Apeiro (2024)

This project was developed in Dr. Daniel Shapiro's CMPM 146 Game AI class taken in Summer 2024. I worked in a team of 4 to develop an AI Visual Novel game where the user can tell the AI to generate any story they want. The LLM model, then, creates a storyline with a climax and resolution, as well as displays images relevant to each scene. The player gets 5 options after each scene: 4 selections or a "write your own" selection where the player can change the trajectory of the story. My contributions included developing frontend, web development features with Three.js to give a more mystical feel to the game. I explored the great possibilites of HTML by creating several webpages (homepage, game selection page, game start page, etc.) to ehance user experience. I learned how to effectively brainstorm ideas and communicate with my team members to produce a high-quality prototype for a game using machine learning. Programming languages used: Three.js and HTML
